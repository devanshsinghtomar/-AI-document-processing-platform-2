# Humanize Document Feature Integration Guide This implementation adds a completely new page called **Humanize Document** to your existing AI Document Processing Platform without changing any existing UI or feature behavior. The feature will: * Convert robotic or AI-generated text into natural human-like content * Preserve original meaning * Keep formatting readable * Work as a separate standalone feature/page * Match your existing UI automatically * Not affect current features --- # STEP 1 — Install Required Package Open terminal in your project root and run: ```bash pip install transformers torch sentencepiece ``` If already installed, no problem. --- # STEP 2 — Create New Backend File Create a new file: ```text backend/humanizer.py ``` Paste this code: ```python from transformers import pipeline import re # Load paraphrasing model only once humanizer_pipeline = pipeline( "text2text-generation", model="google/flan-t5-base", max_length=512 ) def clean_text(text): text = re.sub(r'\s+', ' ', text) return text.strip() PROMPT = """ Rewrite the following text in a natural, human-like, engaging, and readable style. Keep the meaning exactly same. Do not shorten too much. Do not add extra information. Text: """ def humanize_text(text): try: text = clean_text(text) prompt = PROMPT + text result = humanizer_pipeline( prompt, max_new_tokens=300, do_sample=True, temperature=0.8, top_p=0.95 ) generated = result[0]['generated_text'] return generated except Exception as e: return f"Error: {str(e)}" ``` --- # STEP 3 — Update app.py Open your existing: ```text app.py ``` Add this import near other imports: ```python from backend.humanizer import humanize_text ``` --- Now add this NEW ROUTE anywhere below your existing routes: ```python @app.route('/humanize', methods=['GET', 'POST']) def humanize_document(): humanized_output = "" original_text = "" if request.method == 'POST': original_text = request.form.get('text', '') if original_text.strip(): humanized_output = humanize_text(original_text) return render_template( 'humanize.html', output=humanized_output, original_text=original_text ) ``` IMPORTANT: * Do NOT remove or edit any existing route * Only add this new route --- # STEP 4 — Add Navigation Button Open your main navigation template. Usually one of these: ```text templates/index.html ``` or ```text templates/base.html ``` Find navigation/menu section. Add this button/link WITHOUT removing anything: ```html Humanize Document ``` If your project uses buttons instead: ```html ``` --- # STEP 5 — Create New HTML Page Create file: ```text templates/humanize.html ``` Paste this FULL CODE: ```html Humanize Document
